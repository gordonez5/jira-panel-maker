Welcome

JIRA Panel Maker is a simple tool to make it easy to create Acceptance Criteria panels for JIRA.

If this is your first visit, be sure to click on the icon at the top of the screen to learn how to use the tool.

Instructions

Quick start:

  1. Enter number and title
  2. Enter sub criteria text
  3. Reorder list as required
  4. Copy the generated code to the JIRA description

Acceptance Criterion title

To create the panel heading, simply add a criterion number and/or title into the boxes and click the Edit AC Title button.

To edit an existing heading, simply enter new value into the box you wish to change. This will update the heading and the panel output code.

Sub-criteria list

To create an ordered list of criteria, simply enter each one into the text area and click the Add Sub AC button.

The list can then be reordered by dragging the icon up or down.

Deleting items

  • You can delete individual sub-criteria by clicking the icon next to the text.
  • Clicking the Clear all criteria button will delete the entire list. This action can be undone.

Panel code output

Once a list is created, the relevant panel code is generated. This can be viewed in the output tab, labelled with the icon.

Clicking the Copy code button, will copy the markup to your clipboard. This can then be pasted into the description area in a JIRA ticket to display a nicely formatted panel.

Panel demo

To see how the finished panel will look, you can pop up a demo by clicking the View demo button.
